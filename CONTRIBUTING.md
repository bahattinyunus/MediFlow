# Contributing to Apotheca AI

Thank you for your interest in contributing to Apotheca AI (MediFlow)!

## Getting Started

1.  Clone the repository.
2.  Install dependencies for frontend (`cd frontend && npm install`) and backend (`cd backend && npm install`).
3.  Create a new branch for your feature (`git checkout -b feature/amazing-feature`).

## Code Style

-   **Frontend**: We use Prettier and ESLint. Please ensure your code is formatted before committing.
-   **Backend**: Follow standard Node.js best practices.

## Pull Requests

1.  Ensure your code builds and runs locally.
2.  Provide a clear description of your changes.
3.  Link to any relevant issues.
